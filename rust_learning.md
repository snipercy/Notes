## Getting Started

第一次学习 rust 时装的rust是1.3版本，现在(201607)已经是1.10，
先 **卸载**：
``` $ sudo /usr/lib/rustlib/uninstalled.sh ```

** 安装 **
``` $ curl -sSf https://static.rust-lang.org/rustup.sh | sh ```

** 查看版本 **
``` $ rust -V```

** 文档 **
``` /usr/local/share/doc/rust ```

### Hello World

项目管理和工程构建工具：cargo
``` $ cargo new hello ```
