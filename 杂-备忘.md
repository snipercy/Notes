## ssh 快速登录方法

### 方法一
1. 一路回车在当前用户的根目录下的.ssh目录生成id_rsa.pub公钥
```ssh-keygen -t rsa```

2. 将本地生成的公钥拷贝到远程服务器的下并改名为authorized_keys
```scp ~/.ssh/id_rsa.pub 用户名@你的服务器的ip:~/.ssh/authorized_keys```

### 方法二
```
sudo apt-get install sshpass
sshpass -p '你的密码' ssh 用户名@服务器ip地址
```
